### Pour lancer le projet en Dev :

⋅⋅* Installer le CLI expo :
```console
npm install --global expo-cli
```

⋅⋅* Installer le projet :
```console
npm install
```

⋅⋅* Démarrer le projet :
```console
expo start
```

⋅⋅* petit tips en cas de soucis vider le cache :
```console
expo r -c
```
