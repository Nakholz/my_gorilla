###Pour lancer le projet en Dev :

⋅⋅* il vous faut tout d'abbord le cli expo :
```console
npm install --global expo-cli
```

⋅⋅* puis installer le projet :
```console
npm install
```

⋅⋅* enfin lancer projet :
```console
expo start
```

⋅⋅* petit tips en cas de soucis vider le cache :
```console
expo r -c
```
